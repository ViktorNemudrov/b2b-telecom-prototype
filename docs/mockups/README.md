# Макеты для вёрстки (имена = договор с ТЗ)

Сюда кладите **исходные файлы** с **точными именами** из ТЗ. По ним же ориентируемся при реализации UI.

## Правила

- Один файл **`расшифровка.png`** — общий референс и для карточки звонка с кнопкой «Расшифровка», и для экрана полного диалога (одинаковая визуальная система).
- Расширения и регистр имени — как в списке ниже (включая кириллицу и пробелы).
